How to Organize and Analyze Research Data

This module is for students conducting original research. It walks you through formatting data for analysis, running descriptive statistics in Excel, and setting up inferential tests in PASW.

Format Data for Analysis

Use Microsoft Excel and ensure variable names are unique and intuitive. Reformat values as needed for statistical clarity.

Tip: Name your variables clearly and keep data consistent to avoid confusion later.

Make Variables Dichotomous

Convert categorical values like gender or tutorial type into columns with "0" and "1" values. This allows better statistical testing.

Run Descriptive Statistics

Using Excel's Analysis ToolPak, generate summary statistics like mean, median, and standard deviation for continuous values.

Insight: Summary statistics help you see patterns or inconsistencies in your data before running deeper tests.

Run Inferential Statistics in PASW

Import your Excel data into PASW (SPSS) to run tests like Kruskal-Wallis. Choose the right test based on your variables.

Reminder: Save your PASW results in PDF or Word format for documentation and reporting.
